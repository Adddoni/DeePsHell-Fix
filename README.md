# DeePsHell
DeePsHell - это не троян! Это инструмент для обхода DPI-блокировок, который изменяет трафик особым образом (не как VPN). Антивирусы могут фальшиво детектировать его как угрозу из-за методов работы с сетевым трафиком. Я сам лично пользуюсь этим инструментом.

Для пользователей Dr.Web: в защите ПК отключайте Spider Gate, так как он некорректно работает вместе с Zapret, и DeePsHell является его подвидом.

## Прочий функционал
Очистка — это функционал, встроенный в GENERAL, который просто чистит папку Temp, немного не всё, что там есть, а то, что может повлиять на что-либо.

Проверка портов — это вещь, которая создана по большей части для отладки, и вы ей тоже можете пользоваться.

Тест соединения — это вещь, скажет вашу пропускную способность трафика и информацию о вашем соединении. ЕГО ОТКРЫВАТЬ ТОЛЬКО В 1 НА 1.

# Версия v1

### Обозначения для фиксов
Minecraft — этот фикс для Modrinth и CurseForge.

Discord — это фикс для Discord.

Telegram — фикс для Telegram.

Messenger — этот фикс для Discord, Google Meet, Telegram, WhatsApp, Viber, Signal, Snapchat. FaceTime звонки тоже, так что можете позвонить кому-нибудь с ПК и сказать, чтобы скачали это, и устроить созвон в ТГ/ДС.

YouTube — этот фикс просто база, которая всегда будет нужна, и это же фикс для YouTube.

Roblox — фикс для Roblox.

### Обозначения для General
ALL — это фикс, который работает для большинства всех провайдеров. Он работает в большинстве случаев. Если он не будет работать, то сообщите об этом, и тот фикс, который работает в обычном Zapret, будет на его основе сделан специально для тех самых пары процентов людей.

nGa — это вариант, где нет приложений Steam, Battlefield, Roblox, Ubisoft. Почему такой есть? Потому что трафик, проходящий через DeePsHell, будет медленнее примерно на 10–15%, что не очень критично, пинг может быть чуть выше.

Base — этот вариант не хочет видеть какие-то лишние приложения, так что он почти такой же, как и в обычном Zapret, только он ещё включает в себя больший фикс Cloudflare и Telegram, и те же остались для YouTube, Discord.

nCl — это фикс, в котором вырезан Cloudflare, так как из-за него могут быть проблемы.

Hide — это дополнение к обычному fix, только он скрывается и виден только в диспетчере задач.
