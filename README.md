# DeePsHell
DeePsHell - расширенная версия обхода блокировок. Фиксы для мессенджеров (Discord, Telegram, WhatsApp, Viber, Signal, Snapchat, FaceTime) и YouTube. Варианты GENERAL: ALL (основной), Hide (скрытый). Дополнительно: очистка Temp, проверка портов, тест соединения. Антивирусы могут ложно определить как угрозу из-за изменения трафика.
