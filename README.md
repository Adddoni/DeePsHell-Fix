# ОБЗНАЧЕНИЯ ДЛЯ ФИКСОВ

Minecraft - этот фикс для Modrinth и CursForge.

Discord - это фикс для Discord.

Telegram - фикс для Telegram.

Messenger - этот фикс для Discord,Google Mete,Telegram,WhatsApp,Viber,Signal,Snapchat,FaceTime звонки тоже так что можете
позвонить комунибуть с пк и сказать, а у тебя, что звонки не работают.

Youtube - этот фикс просто база которая будет всегда нужен и это же фикс для Youtube

#ОБОЗНАЧЕНИЯ ДЛЯ GENERAL

ALL - это Фикс который работает для большенства всех провайдеров он работает в дольшенстве случаев.
если он будет не работать то сообщите об этом и тот фикс который работает в обычном Zapret и он будет на его
основе сделал спицыально для тех самых пару процентов человек.

nSt - это выреант где нету приложения Steam почему такой есть так как трафик проходяший через DeePsHell
будет медленей примерно на 10%-15% что не очень критично.

Base - этот выреант который не хочет видить какие то лишние приложения так что он почти такой же как и в обычном Zapret только он
ещё и включает в него собственно говоря большей фикс Cloudflare и Telegram, и те же остались для Youtube, Discord

nCl - это фикс в котором чястично вырезан клауд флеер как иза него могут быть проблеммы

Hide - это дополнение к обычному fix только он скрывается и видно только в диспетчере задач

ПРОЧИЙ ФУНКЦИОНАЛ

Очистка - это функционал встроенный в GENERAL который просто чистит папку Temp немного не всё что там есть,
а что может повиять на, что либо

Проверка портов - это вешь которая созданна по большей части для отладки и вы ей тоже можете пользоватся

Тест Соеденения - это вешь скажет вашу пропускню способность трафика и информацию о вашем соеденение ЕГО ОТКРЫВАТЬ ТОЛЬКО В 1 НА 1
